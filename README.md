# 🏔️ PakExplorer — Discover the Soul of Pakistan

**PakExplorer** is a premium, high-performance travel platform dedicated to showcasing Pakistan’s breathtaking landscapes, rich heritage, and diverse cultural experiences. Designed with a visual-first approach, it inspires adventure and simplifies discovery for the world's most stunning hidden gem.

---

## ✨ Features
- **Immersive Discovery:** High-resolution visual storytelling of Pakistan's top destinations.
- **Curated Tour Packages:** Detailed itineraries and comprehensive guides for travelers.
- **Dynamic Interactions:** Smooth, fluid animations and responsive UI components.
- **SEO Optimized:** Built with semantic HTML and modern performance patterns.

---

## 🛠️ Technical Stack

PakExplorer is built using a modern, scalable architecture designed for performance and aesthetic excellence.

### **Core Framework & Build Tools**
*   **[React 19](https://react.dev/):** The latest version of the world's most popular UI library for building dynamic interfaces.
*   **[Vite 8](https://vitejs.dev/):** Next-generation frontend tooling providing lightning-fast Hot Module Replacement (HMR).
*   **[React Router 7](https://reactrouter.com/):** Advanced client-side routing for seamless multi-page navigation.

### **Styling & UI**
*   **[Tailwind CSS 4](https://tailwindcss.com/):** Utility-first CSS framework for rapid UI development with a curated design system.
*   **[Framer Motion](https://www.framer.com/motion/):** Industry-standard animation library for fluid page transitions and micro-interactions.
*   **[Lucide React](https://lucide.dev/):** A collection of beautiful, consistent icons for a modern UI feel.
*   **[Swiper.js](https://swiperjs.com/):** The most modern mobile touch slider for high-impact galleries and carousels.

### **Performance & Utilities**
*   **React Intersection Observer:** For trigger-based animations as elements enter the viewport.
*   **React CountUp:** Dynamic numerical animations for travelers and destination statistics.

## 🎨 Design System

Our aesthetic is inspired by the natural beauty and heritage of Pakistan:

*   **Primary (Indus Emerald):** `#0B5D3A` — Representing lush forests and national identity.
*   **Secondary (Karakoram Snow):** `#F8FAFC` — A clean, crisp white for modern clarity.
*   **Accent (Cholistan Gold):** `#D4AF37` — Representing heritage and golden sunsets.
*   **Typography:** Combining `Playfair Display` for heritage headings and `Inter` for functional UI.

## 🚀 Getting Started

To run PakExplorer locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/shayan9689/PakExplorer.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```
